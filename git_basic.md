- `git config --global user.name {name}`
- `git config --global user.email {email}`
- `git init`
	- 깃 레포 만들기
- `git add {파일 이름}`
	- staging area에 추가
- `git commit -m '{message}'`
	- 커밋(기록) 남기기
- `git status`
	- 상태 확인
- `git log`
	- 커밋 확인
	- `git log --oneline`
- `git remote add origin {url}`
	- 원격 연결
- `git push origin master`
	- 업로드
	- `git push -u origin master`


- `git clone {url}`
	- 깃허브에 있는 깃 새로 가져오기
- `git pull origin master`
	- 이미 연결된 상태에서 당겨오기
